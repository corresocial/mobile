# mobile

**Ambiente de execução**

>Windows 10 Pro x64

>VSCode v1.70.1

>npm v8.5.0

>node v16.14.2

**Dependências globais instaladas**

* expo@57.2.1
* expo-cli@5.4.12
* react-native-cli@2.0.1        
* remove-node-modules@1.7.6     
* yarn@1.22.19

**Execute `npm list -g --depth=0` para listar as dependências globais.**

**Passo a Passo**

* Step 1 - Clone o repositório com o comando: `git clone https://github.com/corre-social/mobile.git ./`
* Step 2 - Ainda no git terminal, alterne para a branch dev com o comando `git checkout dev`
* Step 3 - Abra o VSCode dentro da pasta `/app`
* Step 4 - Execute o comando `npm i` para intalar as dependências
* Step 5 - Reinicie o VSCode, as instalações de tipagem por vezes exigem isso
* Step 5 - Dê um `expo start`

**E Voilà!**
